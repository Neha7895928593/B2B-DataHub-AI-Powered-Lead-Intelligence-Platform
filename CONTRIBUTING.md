# Contributing

Thanks for your interest in improving B2B DataHub.

## Local setup

1. Copy `.env.example` to `.env`.
2. Start the stack with `docker compose up --build`.
3. For local frontend/backend development, use the `Frontend` and `Backend` folders directly.

## Contribution guidelines

- Keep changes focused and easy to review.
- Do not commit secrets or real customer data.
- Update docs when behavior or setup changes.
- Prefer environment-driven configuration over hardcoded values.

## Pull requests

- Explain the problem being solved.
- Include screenshots for UI changes when possible.
- Mention any setup, migration, or environment changes.
